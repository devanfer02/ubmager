# 🔨 Development

1. Clone this repository
2. Run the following command
```zsh
composer install
php artisan key:generate
```

3. Run the application
```zsh
php artisan serve
```

4. Since this project use tailwind, don't forget to run the bellow command
```
npm run dev
```
